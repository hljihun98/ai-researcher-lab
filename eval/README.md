# 평가 하네스

같은 질문에 대한 단일 모델 답변과 AI Researcher Lab 멀티에이전트 최종 답변을
LLM 심판이 블라인드 비교합니다. 심판은 관련성·사실성·실행가능성·완결성을 각각
1~5점으로 평가하며, 답변 순서는 질문 인덱스에 따라 결정적으로 뒤집힙니다.

## 실행

프로젝트 루트에서 API 키를 설정한 뒤 실행합니다.

```bash
python -m eval.harness --n 3 --delay 3
```

특정 모델과 출력 경로를 지정할 수도 있습니다.

```bash
python -m eval.harness --n 6 --delay 5 --model gemini-3.5-flash \
  --output eval/results.json
```

결과는 콘솔 표와 `eval/results.json`에 저장됩니다. JSON에는 질문별 원문 답변,
블라인드 순서, 항목별 점수, 승자와 한 줄 근거 및 전체 멀티 승률이 포함됩니다.

## 비용 및 해석 주의

- 질문 하나당 단일 답변 1회, 멀티에이전트 파이프라인 여러 회, 심판 1회가 필요합니다.
- Gemini 무료 등급에서는 `--n 1`처럼 작게 시작하고 `--delay`를 충분히 두세요.
  전체 벤치마크를 반복 실행하려면 유료 등급을 권장합니다.
- 심판 JSON을 파싱하지 못한 문항은 중립 점수의 `무효` 판정으로 저장되며 승률과
  평균 점수 차이 집계에서는 제외됩니다.
- 데모 모드는 파이프라인 구조 확인용입니다. 캔드 응답이므로 품질 점수에는 의미가 없습니다.

